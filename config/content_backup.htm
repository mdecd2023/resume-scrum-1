.gitignore 檔案內容:

__pycache__ 表示不要將 Python 執行暫存檔案推向遠端

venv/ 表示不要將 Replit 系統中的 virtual environment 目錄推向遠端

config/config 表示不要將設定為近端專用的 config/config 檔案推向遠端
